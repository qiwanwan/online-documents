个人知识库
========================

为什么要做这个知识库
----------------------------

在学习过程中，难免会遇到一些问题。每次遇到问题都要去网上查找资料，实在是太麻烦了。于是我决定把这些问题整理成一个知识库，方便自己和他人查阅。
在这个知识库中，我会记录一些我在学习过程中遇到的问题和解决方案。希望能对你有所帮助。

如何做这样一个在线文档
----------------------------

1. 使用 Sphinx 生成文档
2. 使用 github 进行版本控制
3. 使用 readthedocs 进行在线文档托管
4. 使用 github actions 自动部署

主要内容有哪些
----------------------------
1. 电子电路
2. 编程语言
3. 计算机网络
4. 操作系统
5. 数据库
6. 生活经验
7. 人生感悟 
8. 其他
