PlatformIO 对 ESP-01S 编程注意事项

使用 PlatformIO 通过 Arduino 框架对 ESP-01S 进行编程时，程序无法运行！！怎么办？？？配置文件最后 添加一下两条高亮语句

platformio.ini
1 [env:esp01_1m]
2 platform = espressif8266
3 board = esp01_1m
4 framework = arduino
5 board_upload.resetmethod = nodemcu
6 board_build.flash_mode = dout
