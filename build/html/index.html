个人知识库

2025年5月18日创建

这是什么？

这是一个用 Sphinx + github + readthedocs 搭建的个人知识库，记录了我在学习过程中遇到的一些问题和解决方案。这里的教程都是我亲自操作过的，不会有隐藏的坑。希望能对你有所帮助。

为什么要做这个知识库

在学习过程中，难免会遇到一些问题。很多问题遇到无数次，每次都要去网上查找资料，而且别人写的教程也会存在各种坑，要摸索很多次才会解决问题。有的网站甚至不让复制，实在是太麻烦了。于是我决定把这些问题整理成一个知识库，方便自己和他人查阅。

如何做这样一个在线文档

  1. 使用 Sphinx 生成文档

  2. 使用 github 进行版本控制

  3. 使用 readthedocs 进行在线文档托管

  4. 使用 github actions 自动部署

主要内容有哪些

  1. 电子电路

  2. 编程语言

  3. 计算机网络

  4. 操作系统

  5. 数据库

  6. 生活经验

  7. 人生感悟

  8. 其他
